- **Global Salary Insights**: Compensation data across USA, India, UK, 
  Germany, Canada
- **Personalized Learning**: Curated resources for skill development
- **Interactive Visualizations**: Dynamic charts with Plotly

## Tech Stack
- Python, Streamlit, Plotly, Pandas
- Data visualization and analysis
